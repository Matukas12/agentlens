## 🌟 What is agentlens?

agentlens helps you watch and understand AI agents in action. It provides a clear view of how these agents behave. If you use AI in your projects or are curious about its workings, agentlens shows you what is happening behind the scenes.

## 🔍 What Does agentlens Do?

agentlens offers you tools to:
